# RPG Game
RPG CLI Game made with swift

## Summary

- [Installation](#installation)
- [Usage](#usage)
- [Functions](#functions)

## Installation
```bash
git clone https://github.com/fsuares/rpgcli.git
```

## Usage 

In the main file (main.swift) change variable gameDir to absolute path of location of process

```swift
let gameDir = "/path-to-your-local-repository/"
```

## Functions

 Functional:
- Create Character
- Show Character Stats
- Show Inventory
- Select Missions

Missing:
- Battle Simulator
- Update Character Stats
- Level Progression
